# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-11

### Added
- Initial release of the Soham Trading Corporation Landing Page.
- Interactive 3D Canvas Hero Section representing energy nodes.
- Responsive Navigation with mobile menu overlay.
- "About Us" corporate profile with animated statistic counters.
- Multi-vertical Services Grid (Solar EPC, EV Charging, UPS, Stabilizers, DG Sets, PMC, Electrical Infra).
- Dynamic, filterable Project Portfolio section.
- ROI and Energy Savings Calculators.
- Contact form with WhatsApp integration.
- Glassmorphism design tokens and CSS variables.
- Vite build configuration for optimal production bundling.
- Playwright E2E test suite setup.
- GitHub Actions CI/CD pipeline.
- Dockerfile and docker-compose configurations.
- Complete documentation suite (README, ARCHITECTURE, DEPLOYMENT, SECURITY).
